/shell = Use the run_terminal_cmd tool to execute "./scripts/px4ctl.sh shell" to open an interactive shell in the development container.

## Context
This command opens a bash shell inside the PX4 development container. From here you can:
- Run PX4 commands directly
- Build firmware with `make px4_fmu-v6x_default`
- Access the PX4-Autopilot source code
- Run Python scripts manually
- Debug issues interactively

Type `exit` to leave the shell.
